### Getting started:
```bash
pip install qiskit matplotlib numpy
git clone https://github.com/dmartmillan/quantum-pricing-power.git
python3 main.py
```
### Inputs:
* Option Type (Call or Put)
* Underlying price
* Underlying price volatility
* Risk free rate
* Days to expiration
* Strike
